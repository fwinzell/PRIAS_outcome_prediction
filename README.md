# PRIAS outcome prediction

Code for training and evaluating the models in the paper [Outcome predicition of prostate cancer patients on active surveillance using weakly supervised deep learning](https://www.spiedigitallibrary.org/conference-proceedings-of-spie/13413/134130K/Outcome-prediction-of-prostate-cancer-patients-on-active-surveillance-using/10.1117/12.3046323.full) presented at **SPIE Medical Imaging 2025**

"Outcome prediction of prostate cancer patients on active surveillance using weakly supervised deep learning"
Proc. SPIE 13413, *Medical Imaging 2025: Digital and Computational Pathology*, 134130K (10 April 2025); [https://doi.org/10.1117/12.3046323]
